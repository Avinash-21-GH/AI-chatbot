
# 🎓 AI College Chatbot System

An intelligent AI-based chatbot system developed to provide automated responses to student queries related to college information such as departments, academics, facilities and campus services.

The chatbot uses Natural Language Processing (NLP) techniques along with a web-based interface to enable real-time interaction between users and the system.

------------------------------------------------------------

## 🚀 Features

✔ NLP-based query understanding  
✔ Real-time chatbot interaction  
✔ Automated college information retrieval  
✔ Database integration for storing responses  
✔ User-friendly web interface  
✔ Fast and efficient response system  

------------------------------------------------------------

## 🛠️ Technologies Used

- Python  
- Flask  
- HTML  
- CSS  
- SQLite  
- Natural Language Processing

## ▶️ How to Run the Project

1. Install required libraries

pip install flask

2. Setup database

python setup_db.py

3. Run the application

python app.py

4. Open browser and go to

http://127.0.0.1:5000
